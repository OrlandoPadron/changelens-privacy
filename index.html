Privacy Policy — Changelens for JIRA

Last updated: 2026-06-21

Changelens for JIRA (“the extension”) is a browser extension that shows readable, attributed diffs of JIRA Cloud ticket changes. This policy explains what the extension does and does not do with your data.

Changelens does not collect, transmit, sell, or share any of your data. All processing happens locally in your browser. There is no Changelens server, no analytics, and no third-party data sharing.

What the extension accesses

When you open a JIRA Cloud ticket (a page on *.atlassian.net), the extension runs in that page and reads ticket data using your existing JIRA login (your browser's session cookies). It does not use, request, or store any API tokens, passwords, or OAuth credentials.

Specifically, it reads from your own JIRA instance's API:

  • The issue changelog (field change history) for the ticket you're viewing.
  • The ticket's comments.
  • The ticket's creation date and reporter (for the Time Machine's “original” label).
  • Your own account ID (to optionally distinguish your edits from others').

This is the same data already visible to you in JIRA. The extension only ever requests it from the JIRA site you are already logged into, and only for the ticket you are actively viewing.

What the extension stores

The extension stores a small amount of data locally on your device only, using the browser's chrome.storage.local:

  • “Last viewed” timestamps per ticket, used to highlight what changed since you last looked.
  • Display preferences (e.g. Time Machine step size and “show changes” toggle).

This data never leaves your browser. It is not synced to any server. Uninstalling the extension removes it.

What the extension does NOT do

  • It does not send your data anywhere outside your own JIRA instance.
  • It does not include any analytics, tracking, or telemetry.
  • It does not read pages other than JIRA Cloud (*.atlassian.net).
  • It does not collect personal information for the developer.

Permissions

  • storage — to save your “last viewed” timestamps and display preferences locally (see above).
  • Host access to https://*.atlassian.net/* — so the extension can run on JIRA Cloud tickets and read their change history using your existing session.

Changes to this policy

If this policy changes, the “Last updated” date above will change and the new version will be published at the same location.
